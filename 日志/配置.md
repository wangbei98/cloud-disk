# hdfs相关

* 启动：

```c
$ cd /usr/local/hadoop/sbin
$ start-all.sh
```

* 关闭

```
$ cd /usr/local/hadoop/sbin
$ stop-all.sh
```

* 配置文件

```
cd /usr/local/hadoop/etc/hadoop
```

## flask 相关

* 工作目录

```
$ cd ~/cloud-disk
```

* 启动

```
$ cd backend
$ pipenv shell
$ flask run
```

* 使用gunicorn

```
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

```
gunicorn -w 4 -b 0.0.0.0:5000 app:app -t 3600 -D 
// -t 超时时间
```

* 执行如下命令获取Gunicorn进程树：

```
pstree -ap|grep gunicorn
```
